### 快捷键

* 查看注释 Ctrl + Q

* 类的继承关系 Ctrl + H

* 类的属性、方法 Alt + 7

* Java 转成 Kotlin Ctrl + Shift + Alt + K


### Java

* 变量/参数 下划线

    * Editor -> Language Defaults -> Identifiers -> Reassigned parameters

### Kotlin

* 变量/参数 下划线

    * Editor -> Color Scheme -> Kotlin -> Properties and Variables -> Var (mutable variable, parameter or property)
